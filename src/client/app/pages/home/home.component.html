  • Plug-and-play WebUSB connectivity
  • JavaScript Runtime for Zephyr® OS powered boards
  • Instant JavaScript code execution on target
  • IDE with multiple tabs, console shell, and GitHub integration
  • Web app that works online and offline
  • Integrated documentation and board explorer
